*Movie API*
A simple rest api built using Flask to manage movies.

*Tech Stack*
flask -API
MySql
SQL Alchemy-ORM
Flasgger (Swagger UI)
pytest -unit testing
Git and GitHub

*How to run the project*
1.Clone the repositorygit clone https://github.com/your-username/movie-api.git
cd movie-api
2.Create virtual envirnment python -m venv venv
venv\Scripts\activate
3.Install dependencies pip install -r requirements.txt
4.Run the app python app/main.py
The server will start at:http://127.0.0.1:5000/
swagger:http://127.0.0.1:5000/apidocs/#/
